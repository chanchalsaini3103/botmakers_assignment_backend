# Botmakers Authentication & RBAC System

A full-stack web application with JWT-based authentication and Role-Based Access Control (RBAC) using Spring Boot backend and React frontend.

# Features

-  User Registration with role selection (USER/ADMIN)
-  JWT Token based authentication
-  Role-based access control
-  Separate dashboards for USER and ADMIN roles
-  Protected routes
-  Form validation (Name, Email, Password)
-  Responsive UI
-  Logout functionality

# Tech Stack

# Backend
- Java 17
- Spring Boot 
- Spring Security
- JWT 
- Spring Data JPA
- MySQL Database
- Maven

# Frontend
- React 
- React Router DOM 
- Axios
- React Hook Form
- CSS3
